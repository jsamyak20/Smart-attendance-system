# EduTrack Pro - Academic Attendance System

EduTrack Pro is a modern, responsive, and professional attendance management system designed for schools, colleges, and educational institutions. It features a sleek SaaS-like dashboard, real-time metrics, and a visual calendar overview.

## 🚀 Key Features

- **Professional Dashboard**: Clean overview of daily attendance with interactive metrics.
- **Visual Calendar**: Monthly view showing attendance density and quick-link navigation.
- **Student Management**: Full CRUD (Create, Read, Update, Delete) for student records.
- **Attendance History**: Search and filter past attendance logs by date.
- **CSV Export**: Download attendance reports as spreadsheet-ready CSV files.
- **Dark Mode Support**: Seamless switching between Light and Dark themes.
- **Persistence**: Data is saved locally in your browser's `localStorage`.

## 🛠️ Deployment Instructions

This project is built using vanilla HTML, CSS, and JavaScript. It is ready for deployment on any static site hosting service.

### 1. Netlify / Vercel (Recommended)
Simply upload the root folder or connect your GitHub repository. The project will work immediately out of the box.

### 2. GitHub Pages
1. Push this code to a GitHub repository.
2. Go to **Settings > Pages**.
3. Select the `main` branch and `/root` folder.
4. Click **Save**.

# 📊 Smart Attendance System

The Smart Attendance System is a beginner-friendly web application designed to simplify the process of taking and managing attendance in schools and colleges.

Traditionally, attendance is recorded manually, which can be time-consuming and prone to errors. This project provides a digital solution where attendance can be marked with a single click and stored automatically.

## 🎯 Objective

The main objective of this project is to:
- Replace manual attendance with a digital system
- Reduce time and human errors
- Provide an easy way to track attendance records

## ⚙️ How It Works

- The user logs into the system using a username and password.
- A dashboard displays student details and attendance summary.
- The user can mark students as Present or Absent.
- Attendance data is stored using browser localStorage.
- Records can be viewed anytime based on date.

## ✨ Features

- 🔐 Login Authentication
- 👨‍🎓 Student Management
- ✅ Mark Present/Absent
- 📅 Date-wise Attendance Tracking
- 💾 Data stored in localStorage
- 📊 Attendance Summary
- 🔄 Reset Attendance Option
- 📱 Responsive Design

## 🛠️ Technologies Used

- HTML – Structure of the website  
- CSS – Styling and layout  
- JavaScript – Functionality and logic  
- localStorage – Data storage in browser

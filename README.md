# EduTrack Pro - Academic Attendance System

EduTrack Pro is a modern, responsive, and professional attendance management system designed for schools, colleges, and educational institutions. It features a sleek SaaS-like dashboard, real-time metrics, and a visual calendar overview.

## 🚀 Key Features

- **Professional Dashboard**: Clean overview of daily attendance with interactive metrics.
- **Visual Calendar**: Monthly view showing attendance density and quick-link navigation.
- **Student Management**: Full CRUD (Create, Read, Update, Delete) for student records.
- **Attendance History**: Search and filter past attendance logs by date.
- **CSV Export**: Download attendance reports as spreadsheet-ready CSV files.
- **Dark Mode Support**: Seamless switching between Light and Dark themes.
- **Persistence**: Data is saved locally in your browser's `localStorage`.

## 🛠️ Deployment Instructions

This project is built using vanilla HTML, CSS, and JavaScript. It is ready for deployment on any static site hosting service.

### 1. Netlify / Vercel (Recommended)
Simply upload the root folder or connect your GitHub repository. The project will work immediately out of the box.

### 2. GitHub Pages
1. Push this code to a GitHub repository.
2. Go to **Settings > Pages**.
3. Select the `main` branch and `/root` folder.
4. Click **Save**.

# 📊 Smart Attendance System

The Smart Attendance System is a beginner-friendly web application designed to simplify the process of taking and managing attendance in schools and colleges.

Traditionally, attendance is recorded manually, which can be time-consuming and prone to errors. This project provides a digital solution where attendance can be marked with a single click and stored automatically.

## 🎯 Objective

The main objective of this project is to:
- Replace manual attendance with a digital system
- Reduce time and human errors
- Provide an easy way to track attendance records

## ⚙️ How It Works

- The user logs into the system using a username and password.
- A dashboard displays student details and attendance summary.
- The user can mark students as Present or Absent.
- Attendance data is stored using browser localStorage.
- Records can be viewed anytime based on date.

## ✨ Features

- 🔐 Login Authentication
- 👨‍🎓 Student Management
- ✅ Mark Present/Absent
- 📅 Date-wise Attendance Tracking
- 💾 Data stored in localStorage
- 📊 Attendance Summary
- 🔄 Reset Attendance Option
- 📱 Responsive Design

## 🛠️ Technologies Used

- HTML – Structure of the website  
- CSS – Styling and layout  
- JavaScript – Functionality and logic  
- localStorage – Data storage in browser  

## 🌍 Real-World Use

This system can be used in:
- Schools
- Colleges
- Coaching classes
- Small organizations

## 🚀 Future Scope

- Integration with database (Firebase/MySQL)
- Face recognition attendance
- QR code scanning
- Export attendance to Excel/PDF
- Multi-user login system

## 👨‍💻 Conclusion

The Smart Attendance System is a simple yet effective project that demonstrates how web technologies can be used to solve real-world problems. It is ideal for beginners and can be further enhanced into a full-scale application.
